Aqui faremos uma breve explanação sobre Reset, Reboot e Normalize!

1.	Reset

Zera quase todos os estilos padrões do navegador.  Inicia o layout do zero, sem interferência do navegador. Como vantagem apresenta controle total sobre o design e sua desvantagem seria a necessidade de reestilizar o estilo.

2.	Normalize

Padroniza os estilos entre navegadores, mantém estilos úteis (como tamanho de headings), corrige inconsistências entre os navegadores, ajusta elementos HTML entre diferentes versões e melhora acessibilidade. Como vantagem apresenta menos trabalho na reconstrução de estilos básicos.

3.	Reboot

 É uma mistura de Reset + Normalize, uma base mais profissional, criado pelo Bootstrap, remove estilos ruins, mantém estilos úteis, promove melhorias de tipografia, define padrões modernos (box-sizing, line-height, etc.).
